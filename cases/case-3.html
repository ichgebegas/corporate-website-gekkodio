Переводчик богослужебных чтений

Инструмент для Panteon: помогает готовить переводы чтений, хранить изображения и быстро обновлять тексты для публикации.

КлиентPanteon
Срок3 дня
Год2026
СтекLovable · n8n · OpenAI · Supabase

Задача

Panteon нужно было быстро собрать рабочий инструмент для подготовки богослужебных чтений: вводить итальянский текст, добавлять изображения, переводить фрагменты и хранить готовые материалы в одном месте.

Главное требование - не делать тяжёлую админку с нуля, а за несколько дней получить понятный процесс, который можно поддерживать и дополнять без разработчика.

Решение

Собрали интерфейс на Lovable, подключили Supabase для хранения материалов, OpenAI для помощи с переводом и n8n для служебных сценариев. Получился компактный рабочий инструмент без лишних экранов.

  • Панель ввода чтений: первая, вторая lectura и Евангелие
  • Загрузка и хранение изображений к каждому блоку
  • AI-помощь для чернового перевода и вычитки текста
  • Публичный режим просмотра подготовленных материалов
  • База материалов в Supabase с возможностью дальнейшего расширения

Результат

За 3 дня Panteon получил рабочий переводчик и панель подготовки материалов. Команда может обновлять тексты, добавлять изображения и публиковать чтения быстрее, не собирая процесс вручную из разных файлов.
