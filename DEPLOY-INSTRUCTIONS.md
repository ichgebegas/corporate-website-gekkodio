# Инструкция по запуску и деплою

## Локальный запуск

1. Установите Node.js.
2. В папке проекта выполните:

```powershell
npm install
npm start
```

3. Откройте сайт в браузере:

```text
http://localhost:5501
```

## Заявки

Сейчас форма заявки работает в режиме заглушки: пользователь видит успешную отправку, но сообщение никуда не уходит.

## Деплой на Vercel

1. Установите Vercel CLI:

```powershell
npm i -g vercel
```

2. Авторизуйтесь:

```powershell
vercel login
```

3. Задеплойте проект:

```powershell
vercel --prod
```

После деплоя Vercel покажет production-ссылку на сайт.
